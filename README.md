## Ответы на вопросы
- **Что делает fetch?**  
  `fetch` — это современный способ делать HTTP-запросы (например, GET к API). Возвращает Promise.

- **Зачем нужны async/await?**  
  Чтобы писать асинхронный код так, будто он синхронный. Читаемость ↑, ошибку ловим try/catch.

- **Как работает localStorage?**  
  Это объект в браузере, хранит строки "ключ-значение" до 5-10 МБ. Данные остаются после перезагрузки.

- **Где помог ИИ, а где разбиралась вручную?**  
  ИИ дал основу переключателя тем и загрузки цитат. Вручную: адаптировала под свой дизайн, исправила баги с модалкой после динамической галереи, настроила Picsum вместо Unsplash (Unsplash требует ключ).
